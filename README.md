# zsh-setup UI

Showcase website for [zsh-setup](https://github.com/alivinshiva/zsh-setup) — a portable Oh My Zsh + Powerlevel10k configuration.

Built with Vite + React. Dark/light theme with a TUI-inspired aesthetic.

## Dev

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Output goes to `dist/`.
